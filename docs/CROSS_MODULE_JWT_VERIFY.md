# Cross-Module JWT Verification Standard

To verify tokens generated by the Auth Module securely without making a network call to `/api/auth/verify` on every request, you can decode the JWT locally using the shared `JWT_SECRET` environment variable.

## Setup
1. Ensure your module's `.env` file contains the exact same `JWT_SECRET` generated by the Auth Module.
2. Install dependencies: `pip install fastapi python-jose pydantic`

## FastAPI `Depends()` Snippet
Copy and paste this snippet into your FastAPI project. You can then use `Depends(verify_jwt_token)` on your protected routes.

```python
import os
from fastapi import Depends, HTTPException, status
from fastapi.security import OAuth2PasswordBearer
from jose import JWTError, jwt
from pydantic import BaseModel
from typing import Optional

# Using the standard environment variable
JWT_SECRET = os.environ.get("JWT_SECRET")
ALGORITHM = "HS256"

if not JWT_SECRET:
    raise ValueError("JWT_SECRET environment variable is not set!")

oauth2_scheme = OAuth2PasswordBearer(tokenUrl="https://<auth-module-url>/api/auth/login")

class TokenPayload(BaseModel):
    user_id: int
    email: str
    role: str
    username: str

def verify_jwt_token(token: str = Depends(oauth2_scheme)) -> TokenPayload:
    """
    Dependency to verify the JWT token locally.
    Raises 401 Unauthorized if the token is invalid or expired.
    """
    credentials_exception = HTTPException(
        status_code=status.HTTP_401_UNAUTHORIZED,
        detail="Could not validate credentials",
        headers={"WWW-Authenticate": "Bearer"},
    )
    
    try:
        # The Auth module signs the token with JWT_SECRET and HS256
        payload = jwt.decode(token, JWT_SECRET, algorithms=[ALGORITHM])
        
        email: str = payload.get("email")
        if email is None:
            raise credentials_exception
            
        return TokenPayload(**payload)
        
    except JWTError:
        raise credentials_exception
```

## Example Usage

```python
from fastapi import APIRouter, Depends

router = APIRouter()

@router.get("/protected-data")
def get_protected_data(user: TokenPayload = Depends(verify_jwt_token)):
    return {"message": f"Hello {user.email}, your role is {user.role}"}
```
