# Cross-Module JWT Verification Standard

Other modules can verify tokens generated by the Auth Module securely in one of two ways:

1. **Option 1: Local Verification (Recommended for speed)**: Decode the JWT locally using the shared `JWT_SECRET` environment variable.
2. **Option 2: Centralized Verification (Recommended for strictness)**: Query the Auth Module's `POST /api/auth/verify` endpoint to check if the session is still active/valid in the DB (supports logout revocation check).

---

## Option 1: Local Verification (Fast & Stateless)

Ensure your module's environment contains the exact same `JWT_SECRET` as the Auth Module.

### FastAPI `Depends()` Snippet

```python
import os
from fastapi import Depends, HTTPException, status
from fastapi.security import OAuth2PasswordBearer
from jose import JWTError, jwt
from pydantic import BaseModel

# Using the standard environment variable
JWT_SECRET = os.environ.get("JWT_SECRET")
ALGORITHM = "HS256"

if not JWT_SECRET:
    raise ValueError("JWT_SECRET environment variable is not set!")

oauth2_scheme = OAuth2PasswordBearer(tokenUrl="https://<auth-module-url>/api/auth/login")

class TokenPayload(BaseModel):
    user_id: int
    email: str
    role: str
    username: str

def verify_jwt_token_local(token: str = Depends(oauth2_scheme)) -> TokenPayload:
    credentials_exception = HTTPException(
        status_code=status.HTTP_401_UNAUTHORIZED,
        detail="Could not validate credentials",
        headers={"WWW-Authenticate": "Bearer"},
    )
    try:
        payload = jwt.decode(token, JWT_SECRET, algorithms=[ALGORITHM])
        email: str = payload.get("email")
        if email is None:
            raise credentials_exception
        return TokenPayload(**payload)
    except JWTError:
        raise credentials_exception
```

## Option 2: Centralized Verification (Checks Revocation)

Query the Auth Module's API to ensure the token has not been revoked (via logout).

### FastAPI `Depends()` Snippet

```python
import os
import httpx
from fastapi import Depends, HTTPException, status
from fastapi.security import OAuth2PasswordBearer
from pydantic import BaseModel

AUTH_SERVICE_URL = os.getenv("AUTH_SERVICE_URL", "http://localhost:8001")
oauth2_scheme = OAuth2PasswordBearer(tokenUrl=f"{AUTH_SERVICE_URL}/api/auth/login")

class TokenPayload(BaseModel):
    user_id: int
    email: str
    role: str
    username: str

def verify_jwt_token_api(token: str = Depends(oauth2_scheme)) -> TokenPayload:
    credentials_exception = HTTPException(
        status_code=status.HTTP_401_UNAUTHORIZED,
        detail="Could not validate credentials",
        headers={"WWW-Authenticate": "Bearer"},
    )
    try:
        with httpx.Client() as client:
            response = client.post(
                f"{AUTH_SERVICE_URL}/api/auth/verify",
                json={"token": token}
            )
            if response.status_code != 200:
                raise credentials_exception
            
            data = response.json()
            if not data.get("valid"):
                raise credentials_exception
                
            return TokenPayload(**data.get("payload"))
    except httpx.RequestError:
        raise HTTPException(
            status_code=status.HTTP_503_SERVICE_UNAVAILABLE,
            detail="Authentication service is currently unavailable"
        )
```

## Example Usage

```python
from fastapi import APIRouter, Depends

router = APIRouter()

@router.get("/protected-data")
# Swap verify_jwt_token_local with verify_jwt_token_api if revocation check is needed
def get_protected_data(user: TokenPayload = Depends(verify_jwt_token_local)):
    return {"message": f"Hello {user.email}, your role is {user.role}"}
```
